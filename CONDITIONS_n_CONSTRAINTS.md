# CONDITIONS_n_CONSTRAINTS — 61_Bortovoy_bot_app

## Ресурсы

- **Исполнитель:** Виталий Жандаров (владелец канала) + AI-ассистент (Cursor)
- **Инструменты:** BotFather, GitHub, GitHub Pages, Telegram Desktop + Mobile для проверки
- **Бюджет:** 0 — GitHub Pages бесплатный, бот бесплатный

## Сроки

- Технический деплой (HTML + Pages): выполняется в рамках текущей сессии
- Настройка бота в BotFather: зависит от пользователя (ручная операция в Telegram)

## Ограничения

| Тип | Ограничение |
|-----|-------------|
| Технические | Telegram открывает только HTTPS; HTTP не работает |
| Telegram | Кнопка в шапке канала появляется только если бот добавлен администратором канала |
| Telegram | Mini App кнопка настраивается через BotFather — ручная операция, AI не может выполнить |
| GitHub Pages | Только статический контент (HTML/CSS/JS); серверный код не поддерживается |
| Именование бота | Username бота в Telegram должен оканчиваться на `bot` и быть уникальным глобально |

## Зависимости

- GitHub-аккаунт пользователя должен быть активен и авторизован через `gh`
- Бот должен быть создан в BotFather до настройки Mini App
- Бот должен быть добавлен администратором в @bortovoyzhurnal

## Риски

| Риск | Вероятность | Действие |
|------|-------------|----------|
| Username бота уже занят | средняя | Подобрать альтернативный username |
| Кнопка не отображается в канале | низкая | Проверить, что бот добавлен как admin; переоткрыть канал |
| GitHub Pages не активировался | низкая | Проверить настройки репозитория (Settings → Pages) |
