# CONTEXT — 61_Bortovoy_bot_app

## Предпосылки

Канал @bortovoyzhurnal («Виталий Жандаров | Бортжурнал "Ключ к реальному управлению"») существует и ведётся. Подписчики канала не имеют удобного способа быстро найти услуги Центра, записаться на обучение или консультацию. Информация рассыпана по постам.

Эталон реализации: канал e/acc (@cryptoessay) — кнопка «Инфо & Реклама» в шапке открывает Mini App с навигацией. Скриншоты эталона: [Cursor/technical/telegram-miniapp-example/screenshots/](../technical/telegram-miniapp-example/screenshots/).

## Почему сейчас

Telegram Mini App — стандартный инструмент Telegram (Bot API, BotFather), не требующий сторонних платформ. Доступен без разработки бэкенда: достаточно статического HTML на HTTPS. Технология зрелая, хостинг на GitHub Pages бесплатный.
