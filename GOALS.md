# GOALS — 61_Bortovoy_bot_app

## Главная цель

Добавить в канал @bortovoyzhurnal кнопку «Услуги центра», открывающую Mini App с навигацией по услугам Центра Виталия Жандарова.

## Ожидаемые результаты

- Кнопка «Услуги центра» отображается в шапке канала (десктоп и мобильный)
- Mini App открывается и показывает 4 раздела: Обучение, Проекты, Консультации, Управленческий спецназ
- Приложение приветствует пользователя по имени из Telegram
- HTML задеплоен на GitHub Pages с HTTPS

## Критерии успеха

- Кнопка видна в шапке @bortovoyzhurnal на мобильном и десктоп-клиенте Telegram
- Нажатие открывает Mini App корректно (без ошибок, без HTTP предупреждений)
- Все 4 пункта навигации отображаются

## Anti-goals

- Не нужен бэкенд или серверная логика — только статический HTML
- Не нужна интеграция с CRM или платёжными системами на этом этапе
- Не нужно собирать данные пользователей
